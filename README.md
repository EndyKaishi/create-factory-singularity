# Create: Factory Singularity

Create: Factory Singularity is an expert-level automation modpack built around the philosophy that **everything must be automatable**.

If a resource exists, it can be farmed.
If an item is craftable, it can be fully automated.
If a material is rare, there is a system to scale it.

This is not a kitchen-sink pack.
This is a factory design challenge.

---

## Core Philosophy

Factory Singularity is built on three principles:

### 1. No Dead Ends
No resource is permanently manual.
No material is locked behind RNG forever.
Every progression wall has a logical automation path.

### 2. Create Is the Language
Automation is driven primarily by the Create mod:
- Rotational power
- Mechanical crafting
- Sequenced assembly
- Large-scale production lines
- Train-based logistics

Other mods support progression — they do not replace Create.

### 3. Automation Is the Endgame
Late-game content is not a grind.
It is an engineering problem.

The goal is to build self-sustaining, infinitely scaling factory systems.

---

## Pack Identity

This is an expert automation pack designed for players who enjoy:

- Designing production chains
- Optimizing throughput
- Solving complex crafting loops
- Scaling rare resource generation
- Building monumental factory infrastructure

If you enjoy theorycrafting and system design, this pack is for you.

---

## Major Design Goals

- All rare materials can be automated.
- All progression is factory-solvable.
- Mechanical crafting is central to high-tier progression.
- Late-game content (including extreme-tier crafting) is designed around scalable production.

---

## What This Pack Is NOT

- Not a casual progression pack.
- Not a magic-focused pack.
- Not a passive resource generator experience.
- Not a "place a single block and win" modpack.
